# Zeqing Zhang Personal Website

This is a simple GitHub Pages personal website template.

## How to publish on GitHub Pages

1. Create a repository named `Leeeemon87.github.io`.
2. Upload `index.html` and `style.css` to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for GitHub Pages to deploy.

Your site will be available at:

`https://Leeeemon87.github.io/`

## Edit content

- Update personal information in `index.html`.
- Update visual style in `style.css`.
